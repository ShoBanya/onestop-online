# オンライン行政処理
さっぴー川原（@sapi_kawahara）と申します、わたしは行政に関係する人ではないですが、社会システムのオンライン化を含めて、社会はオンライン化であるべきという考えのもと、自ら進んで新しいサービスを試すのが趣味にしております。

2020年のコロナ蔓延による緊急事態宣言に伴い、外出自粛や在宅勤務になるなどから発生した印鑑問題から行政を始めとする政府機関が、急にオンライン化を進めるように舵を切ったようになっていますが、行政には元からオンラインで処理ができる基盤を持っていました。

たとえば、フリーランスでおなじみの確定申告では住民基本台帳カードを使用したe-Taxシステムが、2006年ごろ（正式スタート日不明）に開始しています。
2017年1月16日よりマイナポータルサイトが開始し、各種ぴったりサービスなどオンライン行政ができるようになっていきました。

特別定額給付金の受取については、2020年5月1日[^myna1]にマイナポータルサイトから行えるようになりました。（各自自治体によってスタート時期は異なる、5月1日スタートは全1,741市区町村のうち679市区町村）

なお、マイナポータルに接続していない806市区町村もオンライ申請できるように配備していくもようです。

[^myna1]: マイナポータルを通じた特別定額給付金のオンライン申請受付開始について [https://www.soumu.go.jp/main_content/000686087.pdf](https://www.soumu.go.jp/main_content/000686087.pdf)

## オンライン行政の始まりは「住民基本台帳カード」
2003年8月25日から受付を開始した、住民基本台帳ネットワークシステムを利用するためのICカードです。
写真無しのAタイプと、写真有りのBタイプが存在し、Bタイプは身分証明証としても使用できます。

住民基本台帳カードには、公的個人認証サービス（Japanese Public Key Infrastructure）で使用する電子証明書を持っており、住民基本台帳に対して認証サービスを無料で行います。

この電子証明書をe-Taxシステムで利用できるようになってから住民基本台帳カードが普及し始めました。
当初、住民基本台帳カードを読むためには難易度が高く、パソコンにインストールするJavaランタイムのバージョンやブラウザのバージョンなどが細かく指定されていていました。

さらに住民基本台帳カードを作るのに1,000円かかるのと、電子証明書をう読むためのカードリーダーが高価なため便利なのですが使われないシステムとなっていきました。

なお、住民基本台帳カードは2016年1月に新規受付を終了し、有効期限は10年なので2025年いっぱいで住民基本台帳カードは完全に終了します。

## 次に登場したのは「マイナンバーカード」
マイナンバーカードの正式名称は「個人番号カード」です。

2015年10月5日[^number1]からマイナンバー通知カードが全世帯に配布されました。これは確定申告や年末調整などでは必要となる個人番号です。マイナンバー通知カードは身分証明証にはならないのですが、写真などを添付して申請することで、身分証明証となるマイナンバーカードを作成できます。

マイナンバーカードにも住民基本台帳カードと同じように電子証明書が搭載されており、住民基本台帳カードに変わる住民基本台帳ネットワークシステムを利用するためのICカードとなっております。

住民基本台帳カードとマイナンバーカードの違いは、住民基本台帳カードは自治体が発行するのですが、マイナンバーカードは地方公共団体情報システム機構（J-LIS）が全国の自治体から委任を受けて発行しています。

住民基本台帳カードからマイナンバーカードに変わり、利用者的にはあまり変化がないですが、総務省のページ[^hikaku]に記載があります。

マイナンバーカードなってからの違いは以下のとおりです。

1. 申請料金が無料になりました。
2. 顔写真なしが選べなくなりました。
3. 有効期限が10年から、発行日から申請者の10回目の誕生日までになりました。
4. 自治体利用が中心から、社会保障を含めた災害対応など多岐にわたる利用になりました。
5. それにともない、法律が住民基本台帳法からマイナンバー法に変わりました。

[^number1]: マイナンバー（個人番号）のお知らせ個人番号カード交付申請のご案内 [https://www.soumu.go.jp/main_content/000379957.pdf](https://www.soumu.go.jp/main_content/000379957.pdf)

[^hikaku]: 住民基本台帳カードとマイナンバーカードの比較 [https://www.soumu.go.jp/kojinbango_card/03.html#hikaku](https://www.soumu.go.jp/kojinbango_card/03.html#hikaku)

## マイナンバーカードの利点
1. 写真付きなので身分証明書になる。
    * マイナンバー通知カードは身分証明書になりません。
2. オンライン処理を受託している自治体では、日本全国にあるコンビニなどで以下の各種証明書が取得できる。
    * 住民票の写し
    * 印鑑登録証明書
    * 住民票記載事項証明書（受託自治体でも別途対応が必要）
    * 戸籍証明書（受託自治体でも別途対応が必要）
    * 戸籍の附票の写し（受託自治体でも別途対応が必要）
    * 各種税証明書（受託自治体でも別途対応が必要）
3. 確定申告のオンライン化ができる。
4. 引越しをするときに、引越し先の自治体に提出する「転出証明書」が発行されない。
    * 郵送申請も可能ですが、マイナンバーカードの情報が更新できません。
5. 各種行政手続きのオンライン化ができる。（受託自治体でも別途対応が必要）
    * 自治体ポイントの申請
    * 子ども手当の申請
    * 特別定額給付金の申請
6. 健康保険証の代わりになる。
    * 2022年以降対応予定です。

## スマホ時代に対応
以前から住民基本台帳カードやマイナンバーカードを使うことで行政処理のオンライン化はされていましたが大きく変わりだしたのは2017年からで、2017年5月10日[^etax1]にe-Tax利用の簡便化が表明されて、2020年1月6日[^etax2]からはスマホだけでe-Taxシステムから確定申告ができるようになりました。

マイナポータルにおいては、2017年11月30日[^myna2]にAndroidスマホ、2019年10月21日[^myna3]にiPhone7以降を使用してログインして利用できるようになりました。

そして、2020年5月1日[^myna1]に特別定額給付金がマイナポータルから申請できるようになりました。

[^etax1]: 【平成31年１月開始】e-Tax利用の簡便化に向けて準備を進めています [https://www.e-tax.nta.go.jp/topics/topics_290510_kanbenka.htm](https://www.e-tax.nta.go.jp/topics/topics_290510_kanbenka.htm)

[^etax2]: スマートフォンからより便利にe-Taxをご利用いただけるようになりました。（令和2年1月6日） [https://www.e-tax.nta.go.jp/topics/topics_200106_sp.htm](https://www.e-tax.nta.go.jp/topics/topics_200106_sp.htm)

[^myna2]: 2017年11月30日マイナンバーカード読み取り対応Androidスマートフォン単体でのログインが可能になりました。 [https://www.cao.go.jp/bangouseido/myna/index.html#mynaPortalUnyo](https://www.cao.go.jp/bangouseido/myna/index.html#mynaPortalUnyo)

[^myna3]: 2019年10月21日マイナンバーカード読取対応スマートフォンにiPhone（7以降）が追加 [https://www.cao.go.jp/bangouseido/myna/index.html#mynaPortalUnyo](https://www.cao.go.jp/bangouseido/myna/index.html#mynaPortalUnyo)

## マイナンバーカードを取得する
マイナンバーカードを持っておくと便利なのでマイナンバーカードをつくりましょう！今はマイナンバーカードを申請するのは不要不急なので申請するはオススメしません。

詳しくはマイナンバーカード総合サイト [https://www.kojinbango-card.go.jp/](https://www.kojinbango-card.go.jp/)をごらんください。

以下にわたしが申請したときの手順と注意点を記載します。

### マイナンバーカードの申請手順
1. 申請サイト[^number2]にアクセスする。
    * マイナンバー通知カードに添付されているQRコードでアクセスすると早いです。
2. 利用規約に同意する。
3. メールアドレスを登録する。
    * QRコードでアクセスすると申請書IDは入力済みとなります。
4. 登録したメールアドレスに本登録用のURLが送信されるのでアクセスします。
    * 注意！たとえばメールアドレスを登録したときがPCで、スマホで登録URLにアクセスしようとすると、利用している端末が異なるよってことで端末認証番号がメールで送信されますので、認証しないと先に進めません。（スマホが一緒でもブラウザが異なるとハマる）
5. 顔写真を登録する。
6. 申請情報を登録します。
    * 注意！「電子証明書発行」の項目があります、給付金オンライ申請には電子証明書が必須なので、電子証明書発行を希望してください。
7. 登録が完了すると、完了メールが届きます。

[^number2]: オンライン申請してみよう [https://net.kojinbango-card.go.jp/](https://net.kojinbango-card.go.jp/)

手順が多いように感じますが、スマホだけで行うと非常に簡単です。

顔写真はスマホの前面カメラの画質で問題ありません。わたしはマイナンバーカードがスタートしたときに申請しており、iPhone 6 Plusの前面カメラの1.2Mピクセルでした。

### マイナンバーカードの受取手順
受取手順はマイナンバーカード総合サイトに掲載[^number3]があるのでご確認ください。
以下にわたしが受取したときの手順と注意点を記載します。

1. マイナンバーカードができましたという交付通知書（葉書）がきます。
2. 役場にて受取日時を予約する。
3. 役場で暗証番号の設定をする。
4. 受取完了です。

[^number3]: 受取にいこう [https://www.kojinbango-card.go.jp/uketori/](https://www.kojinbango-card.go.jp/uketori/)

マイナンバーカードは受取が少し面倒です。申請してから受取可能になるまで最低でもひと月かかります。進捗状況表示などはありません、ただただ待つだけです。

そして、マイナンバーカードは郵送されなく役場にて受取に行かないといけません。そのため、今は不要不急と思いますので、マイナンバーカードの申請はオススメしておりません。役場なので基本的に平日のみの対応で、働いている人には難易度が高いかと思います。

役場で受取るとき暗証番号設定を行いますが、その作業に約30分かかり、その間に役場の人とパソコン1台を専有するので遅刻厳禁です！！

### 特別定額給付金をオンライン申請した感想
特別定額給付金をマイナポータルでオンライン申請した感想です。オンライン申請はポジティブな感想が多く、ネガティブに感じることが無かったです。

1. とにかく手軽、2020年5月1日の朝に申請したのでサイトも軽かったです。
2. 記入する内容が少ない。
    * 申告者氏名、申告者生年月日、住所、連絡先、メールアドレス、扶養家族の氏名、銀行口座と必要最低限でした。
3. 扶養家族の生年月日およびマイナンバーを入力する必要がないです。
4. 印鑑不要。
5. 記入の控えがメールで送られる。

唯一のネガティブに感じたのは電話番号が、各区切りで分割されて入っているので表計算ソフトで開くと、局番の最初の0が消えて表示されてしまいます。その項目を選択して文字列に設定すれば再び表示されますけど、1つにまとめてハイフン区切りを入れておけば良かった気がします。

2. タイムアウトする。
    * 混雑によるものなので時間をおいてやりましょう。
3. 電子証明書の検証に失敗しました お住まいの市区町村の窓口で電子証明書を更新してください。(El123-2303) - (MPA-lS-l05)と表示される。
    * 混雑によるものなので時間をおいてやりましょう。[^trouble3]
4. 住所が異なるエラーについてです。
    * 転居後にマイナンバーカードの電子証明書を更新していない。
    * 市区町村合併による住所変更があった。
        * これらは、役場でマイナンバーカードの電子証明書の更新が必要です。
    * 字、大字表記については、正式な表記にするのが良いと思います、マイナンバーカードから情報を補完で記載しましょう。
